CREATIVE COMMONS MUSIC

Made with: Next.js 10, PostgreSQL







Role: Designer and Developer

This project was initiated with the aim to promote and distribute music that had been released under Creative Commons licenses. The prevailing websites in this niche lacked a user-friendly interface and interaction at that time. Therefore, this project was designed to address this issue by creating a more modern and sophisticated website that would fill this gap. However, the project could not be fully completed as it was eventually integrated into my other project, melodyflix.com, which had already been operational for several years.

RANDOM QUOTE MACHINE

Made with: React, SCSS

Role: Designer and Developer

The Random Quote Machine is a simple React application that was created as a part of the FreeCodeCamp's front-end libraries certification program. The aim of this application was to provide a simple and user-friendly way for users to access a collection of random quotes. If you require additional information regarding this project, please visit the FreeCodeCamp's dedicated project page.
